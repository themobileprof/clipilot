Termux on Android

Clio is built for coding on your phone — modules run via clio-run-module

Android restricts some syscalls inside Go binaries, so workflow modules execute through clio-run-module and bash — not inside the main clio process.

First run on Termux:

  1. Install: curl -fsSL clipilot.themobileprof.com/clio | sh
  2. Run clio, then setup termux
  3. sync when online to cache automation modules

Lite profile on low-memory devices: sync fetches essentials only; use sync full for the complete catalog.
