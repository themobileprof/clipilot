⚠️ How to run modules from Clio

  1. Inside Clio (>> prompt): use setup termux or download copy_file — Clio downloads YAML from this registry.
  2. Clio prints a clio-run-module … command. Do not paste it at >>.
  3. Type exit, then run the command at your normal shell prompt ($).

On Termux, running clio-run-module inside the REPL can cause SIGSYS: bad system call. The bash script is designed to run outside Clio.
